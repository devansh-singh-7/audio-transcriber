# Audio Transcriber

## Setup
1. Clone the repo
2. Copy .env.local.example to .env.local
3. Fill in required environment variables
4. Run: npm install
5. Run: npm run dev

## Deployment Steps
a. Push code to GitHub  
b. Go to railway.app -> New Project -> Deploy from GitHub repo  
c. Configure required environment variables in Railway dashboard
d. Deploy and test the application
